1. Network Throughput

    *. 네트워크 출력
    *. 얼마나 많은 데이터가 단위 시간 내 목적지에 전달될 수 있는가?
    *. bits per second (bps) / data packets per second
    *. 100 bytes per second = 800 bps

    *. 네트워크 출력은 시간에 따라서 변화한다.
    *. 이런 점 때문에 평균 출력 값을 사용한다. (Average Throughput)
    *. 특별한 이유 없이 출력값이 일정하지 않다면, 패킷 손실이 일어났거나 출력을 담당하는 네트워크 컴포넌트에서 문제가 발생했을 것이다.
    *. 좋은 모니터링 도구는 실시간으로 문제를 모니터링하고 threshold 값의 범위를 넘어가는 경우 알람을 발생하는 방식을 많이 사용한다.
